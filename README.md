## 一款好用、好看、开源、多功能的时钟软件，基于PyQt5编写
软件支持云母、亚克力、Aero效果，作者做Fluent特么真的花了很多精力。。。  
甚至win7也能跑  
对，设置有重影现象  
release1.0有很多链接库问题（v1.1release会修），可以自己fork下来运行：  
    装Python3.8，装上PyQt5 Pywin32 requests zhdate就可以跑了  
注意：默认使用苹方0和15字体，请自己安装，后续会放在release上（或者内置）
